Personal Web Presence

With the modern web platform, it has never been easier to establish a web presence. Services like SquareSpace and Wix, and platforms like Wordpress have eliminated some substantial barriers to entry. However, there is still value in knowing how to build a website from scratch, or at least from a barebones starting point.

In this assignment, you will build your own web pages.

Getting Started

Using a template from Start Bootstrap , create a blog site documenting your spring break (or a fictional version of it). You must fulfill the following requirements:

  • Your blog must have at least three "posts."
  • Your page must have at least two images as HTML elements.
  • Your page must use at least one ordered list and one unordered list.
  • You must change the color palette for your website in the stylesheet.

For extra credit, host your blog online and include a link to your blog in the submission.
